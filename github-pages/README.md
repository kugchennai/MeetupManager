# MeetupManager GitHub Pages Landing

This folder is a fully static landing page, isolated from the app code.

## Files
- `index.html`
- `styles.css`
- `app.js`

## Publish on GitHub Pages

Option 1: Publish from `/docs`
1. Copy this folder contents into `docs/` (or set this folder as your docs source path).
2. In GitHub repository settings, enable Pages from the `main` branch and `/docs` folder.

Option 2: Publish from `gh-pages` branch
1. Push these files to a `gh-pages` branch root.
2. In GitHub repository settings, set Pages source to `gh-pages` branch root.

## Notes
- No Next.js runtime is required.
- Replace CTA links (`GitHub`, `Sponsor`, `Contact`, `LinkedIn`) in `index.html` with final URLs.
